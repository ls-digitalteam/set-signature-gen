Email Signature Creator

Enter your details in the form to create an email signature below. Two signatures will be generated: a main signature containing the Southeast Tech logo, and a reply signature without the logo.

To Save Your Signature on PC:

1. In Outlook, create a new signature by clicking File > Options > Mail > Signatures from the message menu.

2. Select New Signature and name it Main Signature. Copy your new main signature from below and paste it into the Outlook signature editor. Click Save.

3. Repeat the above steps to create a signature named Reply Signature. Copy your new reply signature from below and paste it into the Outlook signature editor. Click Save.

2. Select New Signature and name it Main Signature. Highlight and copy your new main signature from below and paste it into the Outlook signature editor. Click Save.

Repeat the above steps to create a signature named Reply Signature. Highlight and copy your new reply signature from below and paste it into the Outlook signature editor. Click Save.

4. Under "Choose default signature", set the New messages dropdown to use your Main Signature, and the Replies/Forwards dropdown to use your Reply Signature.

To Save Your Signature on Mac:

1. On the Outlook menu, select Preferences > Signatures from the message menu.

2. Select + to add a new signature and name it Main Signature. Copy your new main signature from below and paste it into the Outlook signature editor. Click Save.

3. Repeat the above steps to create a signature named Reply Signature. Copy your new reply signature from below and paste it into the Outlook signature editor. Click Save.

4. Under "Choose default signature", set the New messages dropdown to use your Main Signature, and the Replies/Forwards dropdown to use your Reply Signature.
